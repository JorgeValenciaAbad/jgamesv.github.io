Bootstrap Icons

Paso 1

Para poder poner los iconos de bootstrap se debera de introducir la siguiente linea de codigo dentro de la etiqueta head de la pagina.
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.5.0/font/bootstrap-icons.css" >

Paso 2

Tras realizar el primer paso deberemos de escribir el siguiente codigo, donde queramos pero que este dentro de la estiqueta body. En el siguiente codigo se inserta a la pagina un icono de una casa (Para insertar el icono de cualquier otra cosa debemos de poner el nombre dentro del atributo class de la etiqueta un ejeplo es bi-dash):
Ejemplo del codigo:
<span class="bi-house " ></span>
